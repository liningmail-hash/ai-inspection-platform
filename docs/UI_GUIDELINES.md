# UI 实现约束

Stitch 设计稿采用深色工业数字孪生看板风格。实现时保留视觉语言，但页面结构按真实后台系统收敛，避免只做观赏型大屏。

## 视觉原则

- 背景使用深蓝灰，主体容器使用半透明深色面板和 1px 低亮边框。
- 主色使用青蓝 `#00e5ff`，在线/成功使用绿色 `#02c953`，告警/异常使用橙红 `#ff6d00` 或红色。
- 圆角控制在 2-8px，视频窗口更硬朗，卡片不做过度圆润。
- 信息密度中高，适合运维值班长期使用。
- 数字、经纬度、码率、延迟、GPU指标使用等宽字体。

## 页面取舍

- 保留：驾驶舱、视频监控、AI巡检、告警中心、标注训练、模型管理、无人机巡检、设备管理。
- 补齐：报表统计、系统配置、权限审计在一期作为菜单和基础页面保留。
- 调整：如果设计稿偏无人机大屏，要改成后台工作台布局，左侧导航 + 顶部状态栏 + 可操作内容区。

## 组件要求

- 视频窗口必须包含 LIVE、码率、延迟、AI运行状态、快捷操作。
- 告警必须有等级、状态、证据、处置按钮，不只展示趋势图。
- 无人机页必须同时体现机场状态、固定航线任务、实时视频AI和遥测。
- 标注训练页必须体现数据集、样本、标注质检、训练队列、模型指标。
