# ADR 0001：采用私有云边协同架构

## 状态

Accepted

## 背景

AI巡检平台需要接入大量摄像头和无人机视频流。视频数据体积大、实时性要求高，并且通常涉及园区、工厂、能源站点的安全与合规要求。

## 决策

一期采用私有云边协同架构：

- 中心平台负责业务管理、告警处置、训练闭环、模型发布、报表和权限审计。
- 边缘节点负责视频接入、转码、AI推理、事件缓存和断网补传。

## 理由

- 视频默认不出域，降低合规风险。
- 推理靠近视频源，降低延迟和中心带宽压力。
- 中心平台保持统一治理能力。
- 后续多园区复制时，只需扩展边缘节点和站点配置。

## 影响

- 必须定义稳定的中心-边缘接口。
- 边缘节点需要具备断网缓存和补传能力。
- 模型发布需要支持灰度和回滚。
